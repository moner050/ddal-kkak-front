# 딸깍 (DDAL-KKAK) 📊

> 딸깍해서 얻는 금융 정보, AI 기반 주식 분석 웹 애플리케이션

딸깍은 개인 투자자를 위한 AI 기반 주식 분석 플랫폼입니다. 실시간 시장 현황부터 AI 기반 종목 추천, 뉴스 분석, 공시 정보까지 투자에 필요한 모든 정보를 한 곳에서 제공합니다.

## 🆕 최근 업데이트 (2025-11-25)

### ⚡ 성능 최적화 - Build-time Static Data Generation
- **빌드 타임 데이터 생성**: 백엔드 API에서 데이터를 미리 fetch하여 static JSON 파일로 저장
- **제로 API 호출**: 사용자가 페이지 접속 시 API 호출 없이 즉시 데이터 표시
- **항목별 JSON 파일 분리**: undervalued-stocks, featured-stocks, filings 등 개별 JSON 관리
- **자동 빌드 프로세스**: `npm run build:web` 시 자동으로 최신 데이터 fetch
- **데이터 투명성**: UI에 데이터 기준 날짜 명시 (예: "📅 데이터 기준: 2025년 11월 25일")

### 🎓 초보자 친화적 UI/UX
- **듀얼 모드 지원**: 초보자 모드와 전문가 모드 간 원클릭 전환
- **컨텍스트 가이드**: 각 지표 클릭 시 상세 설명 팝업 제공
- **색상 코딩**: 지표 값에 따른 직관적인 색상 표시 (좋음/보통/주의)

## ✨ 주요 기능

### 📊 시장 현황 (11개 핵심 지표)

투자자가 꼭 알아야 할 11개의 핵심 시장 지표를 실시간으로 제공합니다.

#### 1️⃣ 주요 지수
- **S&P 500** - 글로벌 주식시장 대표 지수
- **코스피(KOSPI)** - 한국 시장 대표 지수

#### 2️⃣ 금리 및 변동성
- **미국 10년물 국채수익률** - 장기 금리 동향 및 경기 전망
- **VIX (변동성지수)** - 시장 공포도 및 변동성 측정

#### 3️⃣ 원자재 및 암호화폐
- **비트코인(BTC)** - 위험자산 심리 및 디지털 자산 동향
- **WTI 원유** - 인플레이션 및 경기 선행지표

#### 4️⃣ 환율 및 안전자산
- **원·달러 환율(USD/KRW)** - 외환 시장 동향
- **금 시세(Gold)** - 안전자산 선호도

#### 5️⃣ 밸류에이션 지표
- **미국 버핏지수** - 미국 시장 밸류에이션 (시가총액/GDP)
- **한국 버핏지수** - 한국 시장 밸류에이션 (시가총액/GDP)

#### 6️⃣ 투자 심리
- **미국 공포·탐욕 지수** - 시장 참여자들의 심리 상태 측정

### 💎 AI 기반 종목 추천

6가지 투자 전략별 맞춤형 종목 추천 시스템을 제공합니다.

#### 🎓 초보자/전문가 모드
- **초보자 모드 (🌱)**
  - 카드 형식의 직관적인 UI
  - 핵심 4개 지표만 표시 (ROE, PER, PEG, 영업이익률)
  - 각 지표 클릭 시 상세 설명 제공
  - 페이지당 12개 종목 표시

- **전문가 모드 (📊)**
  - 테이블 형식의 정보 밀도 높은 UI
  - 13개 전체 재무 지표 표시
  - 정렬 가능한 컬럼 헤더
  - 페이지당 30개 종목 표시

#### 📈 장기 투자 전략
1. **저평가 우량주 (워렌 버핏 스타일)**
   - 높은 수익성과 성장성을 갖춘 기업을 합리적인 가격에 매수
   - PER, PEG, ROE, FCF Yield 등 종합 분석

2. **기본 가치투자**
   - 저평가된 기업을 발굴하는 기본적인 가치투자 전략
   - 시가총액, 영업이익률, ROE 기반 선별

3. **엄격한 가치투자**
   - 더 까다로운 기준으로 우량한 저평가 기업 선별
   - 재무 건전성과 성장성을 동시에 고려

4. **성장+품질 (장타 전략)**
   - 높은 성장성과 품질을 갖춘 기업 장기 보유
   - 매출/EPS 성장률 15% 이상, 높은 영업이익률

#### ⚡ 단기 투자 전략
5. **모멘텀 트레이딩 (단타)**
   - 강한 상승 추세를 보이는 종목 단기 매매
   - RSI, MACD, 거래량 기반 기술적 분석

6. **스윙 트레이딩 (단타)**
   - 적절한 변동성을 가진 종목의 단기 등락 활용
   - ATR 변동성, 볼린저밴드 활용

#### 📊 AI 분석 기능
- **AI 스코어 (0-100점)** - 머신러닝 기반 종목 평가
- **투자 심리 분석** - 긍정/중립/부정 센티먼트 분석
- **신뢰도 점수** - AI 분석 결과의 신뢰도 표시
- **목표가 및 상승여력** - 예상 목표가와 상승 가능성

#### ⚡ 성능 최적화
- **정적 데이터 방식** - 최대 1000개 종목 데이터를 한 번에 로드
- **즉각적인 필터링/정렬** - 서버 호출 없이 클라이언트에서 처리
- **데이터 기준 날짜 표시** - 명확한 데이터 업데이트 시점 안내
- **빠른 페이지 전환** - 페이지 이동 시 지연 거의 없음

### 📰 AI 뉴스 분석

경제/금융 뉴스를 AI로 분석하여 핵심 정보만 빠르게 제공합니다.

- **8개 카테고리별 분류**
  - 거시경제, 금융시장, 부동산, 기업/산업
  - 소비/고용, 정책/제도, 정치, 전체

- **AI 기반 요약**
  - 긴 기사를 3-5줄로 요약
  - 핵심 키워드 추출
  - 시장 영향도 분석

- **실시간 업데이트**
  - 최신 뉴스 자동 수집
  - 중요도 기반 정렬

기업 공시를 실시간으로 모니터링하고 AI가 영향도를 분석합니다.

## 🚀 시작하기

### 필수 요구사항

- Node.js 18.x 이상
- npm 또는 yarn
- **백엔드 서버 실행 중** (또는 환경변수 설정)

### 설치

```bash
# 저장소 클론
git clone https://github.com/moner050/ddal-kkak-front.git

# 디렉토리 이동
cd ddal-kkak-front

# 의존성 설치
npm install
```

### 데이터 준비 (필수)

프로젝트 실행 전 백엔드 API에서 데이터를 fetch하여 static JSON 파일 생성:

```bash
# 백엔드 서버가 http://finance-mhb-api.kro.kr에서 실행 중이어야 함
npm run fetch-data

# 또는 다른 API URL 사용
EXPO_PUBLIC_API_URL=http://your-api-url npm run fetch-data
```

생성되는 파일:
- `public/data/undervalued-stocks.json` (1000개 종목)
- `public/data/featured-stocks.json` (10개 주목 종목)
- `public/data/filings.json` (20개 공시)
- `public/data/metadata.json` (메타정보)

### 개발 서버 실행

```bash
# 웹 개발 서버 시작
npm run web

# 또는
npx expo start --web
```

브라우저에서 `http://localhost:19006` 으로 접속

### 빌드

```bash
# 웹 빌드 (자동으로 데이터 fetch 후 빌드)
npm run build:web
```

**주의**: 빌드 전 `prebuild:web` 스크립트가 자동으로 `fetch-data`를 실행합니다.
